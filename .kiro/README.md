# Kiro Configuration

This directory contains Kiro AI configuration and context files for the ImageZip project.

## Contents

- `steering/`: Contains project context and guidelines for AI-assisted development
  - `project-context.md`: Overview of the project, problem statement, and solution approach

## Purpose

This directory is included as part of the AI for Bharat Week 1 Micro-Tools challenge requirements to demonstrate the use of AI-assisted development tools in building this application.

## Note

This directory should NOT be added to .gitignore as it's required for the challenge submission.
